## ⚙️ Getting Started

To run this project locally:

```bash
git clone https://github.com/efemalis/notemarket.git
cd notemarket
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
GROQ_API_KEY=your_groq_api_key
```

You can also create a `.env.example` file for better developer experience.

## 🧠 Architecture Overview

* **Frontend (Next.js)** → UI & user interactions
* **Supabase** → Authentication, database, and storage
* **AI Layer (Groq API)** → Summarization, Q&A, quiz generation
* **OCR Pipeline** → Converts handwritten/PDF notes into structured text
